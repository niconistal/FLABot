Ejemplos:

- El archivo Sample.txt tiene el ejemplo de un predicado que calcula un número tal que
  su factorial es mayor a un número dado. Editar el archivo para saber que va en cada máquina.

- En el archivo predicates.doc hay otro ejemplo. Es un agente que arma una computadora de acuerdo 
  a ciertas preferencias. Se detalla la información (código prolog) que van en las
  bases de datos de cada máquina (A, B y C), adicionados a la base de datos normal de JavaLog.
  El código del agente es el código Movilog que implementa el agente de búsqueda.
